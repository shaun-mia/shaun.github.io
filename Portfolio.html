Classical Model Dashboard

The Classical-model-Dashboard project utilizes Power BI to develop a robust and interactive dashboard aimed at enhancing data analysis and visualization. This project demonstrates advanced data modeling and visualization techniques, including the use of DAX functions for calculations like cumulative sales and moving averages. It features a comprehensive set of visualizations such as column charts, KPI indicators, and gauge charts, along with interactive elements like slicers for dynamic data exploration. The dashboard is designed to be mobile-friendly, ensuring accessibility on various devices. Key components include data connectors, Power Query Editor tools, and methods for appending, merging, pivoting , and unpivoting data. The project highlights proficiency in Power BI, Python, and SQL, showcasing skills in creating impactful and accessible data visualizations.

E-commarce sales Analysis Dashboard

Developed an interactive Power BI dashboard for an e-commerce sales dataset, featuring key KPIs including total sales amount, profit, quantity, and average order value. Utilized Power Query for data transformation and DAX for advanced calculations. The dashboard includes visualizations such as monthly profit trends, sales amount by category, and detailed insights by customer and payment model. It offers a user-friendly slicer for quarterly data analysis and a comprehensive view of profit by sub-category. This project demonstrates proficiency in data analysis and visualization, providing actionable insights for business decision-making.

Pizza Sales Performance Dashboard

The Pizza Sales Performance Dashboard is a Power BI project designed to analyze and visualize pizza sales data. This interactive dashboard provides valuable insights into sales performance, peak ordering times, and the popularity of different pizzas and categories. By using DAX calculations and custom columns, the dashboard helps stakeholders make data-driven decisions to improve sales strategies and business outcomes.
