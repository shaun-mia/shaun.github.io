Classical Model Dashboard

View Project

The Classical-model-Dashboard project utilizes Power BI to develop a robust and interactive dashboard aimed at enhancing data analysis and visualization. The project highlights proficiency in Power BI, Python, and SQL, showcasing skills in creating impactful and accessible data visualizations.
